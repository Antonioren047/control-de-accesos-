# Manual de usuario: Fase 6

## Vigilante

1. Inicia el turno con conexión mediante QR, PIN y fotografía. Esto autoriza el dispositivo para operar sin conexión.
2. El portal muestra `En línea` o `Sin conexión` y el número de registros pendientes.
3. Si no hay red al registrar la salida, el cierre queda guardado en el dispositivo.
4. Al volver la conexión, la sincronización ocurre automáticamente. También puedes pulsar **Sincronizar**.
5. No borres los datos del navegador mientras existan pendientes.

Las visitas no se validan offline. Para autorizar o registrar un visitante se requiere conexión.

## Supervisor

El menú **Sincronización** muestra conflictos y registros recibidos después de 12 horas. Revisa vigilante, lugar, punto, fecha y UUID; luego selecciona **Aceptar** o **Rechazar** y escribe una justificación de al menos 10 caracteres.

Aceptar o rechazar nunca borra el historial de auditoría.
