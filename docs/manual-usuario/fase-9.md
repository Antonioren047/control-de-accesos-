# Manual de usuario: supervisiones

## Administrador

Abra **Supervisiones** y seleccione **Programar**. Elija lugar, uno o varios puntos, frecuencia y fecha. También puede iniciar una supervisión manual.

## Supervisor

Inicie una supervisión pendiente o manual dentro de su alcance. El detalle muestra el vigilante presente, eventos y recorridos recientes. Use **Tomar evidencia** para capturar una fotografía desde la cámara.

Para finalizar, registre observaciones y confirme su nombre y PIN. El vigilante o responsable confirma también nombre y PIN. Si no se encuentra presente, active la opción de ausencia y documente el motivo. Después del cierre puede abrir el PDF o agregar comentarios, pero no modificar el contenido original.

En datos demo, el PIN de confirmación del supervisor es `102938`. El PIN del vigilante es el de su credencial operativa.
