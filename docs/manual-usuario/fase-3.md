# Fase 3 — Clientes, lugares y residentes

El menú muestra únicamente información dentro del alcance del usuario.

- **Clientes:** permite al Superadministrador crear clientes y consultar su estado. El Administrador consulta sus clientes asignados.
- **Lugares y puntos:** permite administrar o consultar lugares, puntos de acceso y unidades según el rol.
- **Usuarios:** en esta fase permite crear residentes y asociarlos inicialmente con una unidad autorizada.
- **Mis unidades:** muestra al Residente únicamente las unidades relacionadas con su cuenta.

Los códigos aceptan letras, números, guion y guion bajo. Las unidades disponibles son Casa, Departamento, Lote y Nave industrial. Al crear un residente se exige una contraseña temporal robusta y el sistema obliga a cambiarla posteriormente.

Desactivar un registro no lo elimina; conserva su historial para auditoría.
