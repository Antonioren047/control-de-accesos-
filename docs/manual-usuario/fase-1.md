# Manual de usuario — Fase 1

Esta fase solo ofrece panel técnico, instalador y documentación API. No incluye autenticación ni módulos operativos. El instalador solicita base de datos, empresa, superadministrador, zona horaria y URL; al finalizar queda bloqueado. Para desbloquearlo de forma excepcional, un superadministrador técnico debe respaldar la instalación y eliminar manualmente storage/installed.lock.
