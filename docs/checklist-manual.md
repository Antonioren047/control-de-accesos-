# Checklist manual

- [ ] El panel abre en 1440 px, 768 px y 375 px.
- [ ] El menú se colapsa en escritorio y abre como drawer en móvil.
- [ ] Tema claro, oscuro y automático persisten.
- [ ] El instalador rechaza campos vacíos, correo inválido y contraseña débil.
- [ ] Una instalación correcta crea .env, tablas, catálogos, empresa y usuario.
- [ ] Una segunda visita muestra el instalador bloqueado.
- [ ] /api/health responde JSON y no muestra credenciales.
- [ ] /docs muestra Swagger UI.
- [ ] 403, 404 y 500 no muestran trazas en producción.
